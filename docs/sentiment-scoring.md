# Senticore 감성 채점 체계

## 목적

Senticore의 감성 점수는 고객의 기분을 단순히 긍정/부정으로 분류하는 값이 아니다.
채권 상담에서 다음 응대 전략을 정하기 위한 "상담 가능성 점수"로 사용한다.

이 점수는 다음 판단에 연결된다.

- 지금 유선 상담을 이어갈 수 있는가
- 공감 완충 표현이 먼저 필요한가
- 분할납부 또는 납부일 협의가 가능한가
- 반복 독촉보다 절차 확인과 고지 이력 검토가 필요한가
- 관계 유지성 안내로 충분한가

## 기본 출력 형식

AI는 상담 메모 1건을 분석해 아래 JSON을 반환한다.

```json
{
  "score": 0,
  "cooperation": 0,
  "stability": 0,
  "keywords": [],
  "summary": "",
  "strategy": ""
}
```

필드 의미:

| 필드 | 범위 | 의미 |
|---|---:|---|
| `score` | 0-100 | 종합 감성/상담 가능성 점수 |
| `cooperation` | 0-100 | 납부일, 금액, 자료 제출 등 상담 협조 가능성 |
| `stability` | 0-100 | 감정 안정도, 민원/불만/불안 위험의 반대 지표 |
| `keywords` | 배열 | 판단에 사용된 핵심 표현 |
| `summary` | 문장 | 감성 상태 요약 |
| `strategy` | 문장 | 상담원에게 권고할 다음 응대 방향 |

## 점수 구간

| 구간 | 라벨 | 해석 | 권고 응대 |
|---:|---|---|---|
| 0-30 | 부정적 | 상담 거부, 강한 불만, 민원 위험, 납부 의사 낮음 | 공감 우선, 압박 금지, 반복 독촉 제한 |
| 31-60 | 관찰 | 불안정하거나 애매함. 상환 의지는 있으나 확약 부족 | 짧고 중립적인 확인, 납부 가능일/금액 구체화 |
| 61-80 | 협조적 | 상환 협의 가능. 설명과 선택지 제시에 반응 가능 | 유선 상담 또는 분할납부 협의 |
| 81-100 | 매우 협조적 | 정상 납부, 감사 표현, 빠른 회신, 자료 제출 | 관계 유지, 다음 납부일 리마인드 |

현재 앱의 전략 기준:

- `score < 25`: 최우선 위험 고객 후보
- `score < 40`: 공감 우선 완충 톤
- `40 <= score < 61`: 관찰 구간, 중립적이고 짧은 안내
- `score >= 61`: 협조적 톤
- `score >= 75`: 관계 유지 톤 후보

## 채점 원칙

점수는 말투의 친절함만 보지 않는다.
채권 상담에서 중요한 행동 신호를 함께 본다.

예시:

- "힘들지만 다음 주 일부 납부하겠다"는 부정 감정이 있어도 50-65점이다.
- "법대로 해라. 전화하지 마라"는 문장이 짧고 차분해도 0-20점이다.
- "무섭다, 어떻게 되는 거냐"는 불안 표현이므로 `stability`는 낮지만, 상담을 이어갈 수 있으면 `score`는 35-55점이 가능하다.
- 시스템 안내 문자처럼 고객 반응이 없는 이력은 중립값 45-55점으로 본다.

## 가점 기준

| 신호 | 점수 영향 | 설명 |
|---|---:|---|
| 납부 의사 표현 | +10~20 | "납부하겠다", "일부라도 내겠다" |
| 구체적 납부일 제시 | +15~25 | "다음 주 수요일", "월급일 이후" |
| 구체적 금액 제시 | +10~20 | "30만원 가능", "일부 납부 가능" |
| 자료 제출/회신 | +10~20 | 증빙자료, 이메일, 문자 답변 |
| 감사/확인/협조 표현 | +5~15 | "확인했습니다", "감사합니다" |
| 자동이체/향후 일정 유지 의사 | +10~15 | 정상 납부 관계 유지 신호 |

## 감점 기준

| 신호 | 점수 영향 | 설명 |
|---|---:|---|
| 연락 거부 | -25~-40 | "전화하지 말라", 수신 거부, 통화 강제 종료 |
| 납부 불가 단정 | -15~-30 | "못 낸다", "낼 돈 없다"만 반복 |
| 강한 불만/항의 | -20~-35 | 민원, 항의, 비난 표현 |
| 법적 절차에 대한 공격적 반응 | -15~-30 | "법대로 해라", "마음대로 해라" |
| 약속 불이행 반복 | -10~-25 | 이전 납부 약속 미이행 |
| 무응답/회신 없음 | -5~-20 | 안내 후 답변 없음 |
| 일정 확약 없음 | -5~-15 | 납부 의지는 있으나 날짜 없음 |

## 보조 점수 기준

### cooperation

협조도는 고객이 상담원이 필요한 정보를 줄 가능성을 본다.

- 80 이상: 납부일, 금액, 증빙, 회신이 명확함
- 60-79: 협의 가능성이 있고 질문에 답함
- 40-59: 일부 반응은 있으나 확약이 부족함
- 20-39: 답변 회피, 반복 설명, 제한적 협조
- 0-19: 연락 거부, 통화 종료, 협의 불가

### stability

감정 안정도는 민원/불만/불안 위험의 반대 지표다.

- 80 이상: 차분하고 우호적
- 60-79: 안정적이나 사안 설명 필요
- 40-59: 불안 또는 불만이 있으나 상담 가능
- 20-39: 감정 고조, 반박/불만 반복
- 0-19: 강한 항의, 상담 거부, 민원 위험 높음

## 대표 예시

| 상담 메모 | score | cooperation | stability | 전략 |
|---|---:|---:|---:|---|
| "다음 주 중 일부 납부 가능성을 언급함. 불안정하지만 협조 의사는 있음." | 58 | 65 | 40 | 납부 가능일과 금액을 구체화 |
| "더 이상 전화하지 말라며 통화 종료. 상환 가능일 확인 실패." | 12 | 10 | 20 | 반복 독촉 금지, 고지 이력 확인 |
| "완납 처리 확인에 감사 인사를 전달함. 자동이체 유지 의사 확인." | 94 | 95 | 90 | 관계 유지 안내 |
| "장기 연체 안내 문자 발송. 수신 확인은 되었으나 회신 없음." | 18 | 20 | 45 | 회신 부재 관리, 다음 접촉 예약 |
| "경제적 어려움을 설명했고 납부 의지는 있으나 일정 확약은 보류됨." | 49 | 55 | 45 | 상환 가능 일정 재확인 |

## 고객 최종 점수 산정

현재 앱은 고객의 상담 이력별 `sentimentScore` 평균을 고객 최종 감성 점수로 사용한다.
상담 이력이 없으면 기본값은 50점이다.

운영 단계에서는 단순 평균보다 최근 이력을 더 크게 반영하는 방식이 적합하다.

권장 가중치:

- 최근 1건: 50%
- 이전 2-3건: 30%
- 과거 이력: 20%

단, 연락 거부, 민원 위험, 법적 절차 관련 강한 반응은 최신 이력 가중치를 더 높인다.

## 프롬프트 지침

AI에게 감성 분석을 요청할 때는 아래 원칙을 포함한다.

```text
감성 점수는 고객의 단순 기분이 아니라 채권 상담 가능성 점수다.
납부 의사, 납부일/금액 구체성, 회신 여부, 연락 거부, 민원 위험, 감정 안정도를 함께 판단한다.
경제적 어려움 호소만으로 낮은 점수를 주지 말고, 협조 의사와 구체적 약속이 있으면 중간 이상으로 평가한다.
연락 거부, 강한 불만, 통화 종료, 납부 불가 단정은 강하게 감점한다.
JSON만 출력한다.
```

## 주의사항

- 감성 점수는 법적 조치 여부를 자동 결정하지 않는다.
- 점수는 상담원의 판단을 보조하는 값이며, 확정 승인/감면/법적 절차 단정에 사용하지 않는다.
- 고객에게 점수, 위험 라벨, 내부 전략명을 직접 노출하지 않는다.
- 개인정보, 계좌번호, 주민등록번호 원문을 감성 분석 프롬프트에 넣지 않는다.
