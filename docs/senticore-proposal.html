PROJECT DEFINITION

AI 상담보조 시스템: SentiCore

AI는 기준을 만들고 사람은 판단에 집중합니다
SentiCore는 고객의 감성(Sentiment) 데이터와 금융 코어(Core) 데이터를 결합하여 최적의 상담 가이드라인을 도출하는 지능형 판단 보조 시스템입니다.
상담 화면 진입 시 고객별 핵심 상태 및 대응 전략 실시간 표시
데이터 기반 표준 분류체계를 통한 상담 품질의 상향 평준화
행동 데이터 연계로 악성 민원 및 채권 리스크 사전 경고 시스템 구축
SentiCore Strategy Overview | 기존 데이터 자산 100% 활용 및 폐쇄망 내 보안 확보 기반

STRATEGY

구현 전략 및 유효성 검증 방향

보안 및 인프라 방향

  • 독립적 폐쇄망 구동: 외부 네트워크 차단 환경 내 단독 구동으로 데이터 보안 완벽 보장
  • 금융권 보안 요건 충족: 고객 정보 유출 리스크 없는 안전한 sLLM 연동 체계 구축
  • 가용 자원 최적화: 고사양 GPU 서버 신설 없이 내부 인프라 내 효율적 구동 타진

품질 검증 및 향후 과제

  • 답변 적정성 집중: 기술적 메커니즘보다 생성된 답변의 실무 적합성 검증에 우선 순위
  • 경량 모델 로컬 테스트: 오픈소스 모델을 통한 가이드라인 및 초안 작성 품질 사전 확인
  • 현업 피드백 루프: 실제 상담원의 사용성 평가를 통한 단계적 고도화 및 영점 조절
※ 본 자료는 기획 단계의 예시이며, 향후 정밀 품질 검증(PoC)을 통해 고도화할 예정입니다.
